# Birdsong Environmental Monitoring using Digital Signal Processing

## Overview

This project demonstrates the application of Digital Signal Processing (DSP) techniques for analyzing bird vocalizations and exploring their use in environmental monitoring.

Bird songs contain valuable acoustic information that reflects species diversity, habitat conditions, seasonal changes, and ecosystem health. By analyzing recorded bird vocalizations in both time and frequency domains, meaningful environmental insights can be extracted.

The project was completed as part of the EC5011 – Digital Signal Processing course at the Faculty of Engineering, University of Jaffna.

---

## Objectives

- Record and analyze an environmental acoustic signal.
- Visualize the signal in the time domain.
- Analyze the frequency spectrum using Fast Fourier Transform (FFT).
- Extract Mel Frequency Cepstral Coefficients (MFCCs).
- Investigate the potential of birdsong as a bio-indicator for environmental monitoring.
- Propose a real-world DSP-based application.

---

## Project Concept

Bird vocalizations serve as natural indicators of ecosystem health and biodiversity. Changes in bird activity and acoustic patterns can provide valuable information about:

- Biodiversity levels
- Habitat quality
- Environmental disturbances
- Climate-related changes
- Urban ecosystem conditions

By continuously monitoring bird sounds, environmental changes can be detected passively and efficiently.

---

## Signal Processing Workflow

1. Record bird vocalization signal.
2. Convert audio into WAV format.
3. Load the audio signal into MATLAB.
4. Analyze the waveform in the time domain.
5. Compute the frequency spectrum using FFT.
6. Extract MFCC features.
7. Interpret results for environmental monitoring applications.

## Results

### Time Domain Analysis

The birdsong waveform was visualized to study temporal characteristics such as:

- Activity patterns
- Vocalization duration
- Signal amplitude variations

### Frequency Domain Analysis

FFT analysis revealed dominant frequency components present in the birdsong, which can be used for:

- Species identification
- Habitat classification
- Acoustic environment assessment

### MFCC Feature Extraction

MFCC features were extracted to capture the spectral characteristics of bird vocalizations, which are widely used in:

- Acoustic classification
- Pattern recognition
- Machine learning applications
